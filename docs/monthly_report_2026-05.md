## 3. 현재 아키텍처

```
입력 센서
├── IMU (6ch: Acc XYZ + Gyr XYZ)  →  IMU Encoder (1D-CNN)
├── 족저압 (16×8 그리드)          →  Pressure Encoder (2D-CNN)
└── 기압·지자기 (5ch)             →  MagBaro Encoder (1D-CNN)
                                         ↓
                              Cross-Modal Fusion (Transformer)
                                         ↓
                         ┌───────────────┼───────────────┐
                    질환 분류          낙상 예측        보행 지표
                  (4 클래스)         (binary)        (continuous)
```
