# ShoeAlls AI 보행 분석 알고리즘 최종 성과 보고서

**작성일:** 2026-05-01
**상태:** 최종 기술 검증 완료 (PoC)

## 1. 프로젝트 개요
본 프로젝트는 스마트 인솔(IMU, 족저압)과 외부 카메라(스켈레톤) 데이터를 융합하여 주요 뇌질환(파킨슨, 뇌졸중, 치매 등)을 조기에 탐지하고 보행 상태를 분석하는 멀티모달 AI 시스템 구축을 목표로 함.

## 2. 주요 기술적 성과

### ① 초경량 모델 설계 (Mobile-Ready)
- **기존 모델:** 약 123만 개의 파라미터 (4.7 MB)
- **경량 모델:** 약 **14만 개의 파라미터 (0.5 MB)**
- **성과:** 모델 크기를 **88.6% 축소**하면서도 가상 데이터셋에서 **정확도 100% 유지**. 모바일 기기에서의 실시간 추론 지연 시간을 **8.35ms**로 단축함.

### ② 질환별 정밀 진단 (Fine-tuning)
- **뇌졸중(Stroke):** 편마비로 인한 좌우 비대칭성을 100% 정확도로 탐지.
- **파킨슨병(Parkinson's):** 잘게 써는 걸음 및 떨림 패턴을 99.7%의 높은 신뢰도로 분류.
- **치매(Dementia):** 보행 속도 저하 및 불규칙성을 학습하여 조기 진단 가능성을 입증.
- **범용성:** 11개 이상의 다양한 보행 장애 클래스에 대응하는 전이 학습(Transfer Learning) 체계 구축.

### ③ 모바일 및 임베디드 최적화
- **양자화:** INT8 동적 양자화를 적용하여 모델 용량을 **357 KB**까지 압축 (약 80% 추가 절감).
- **호환성:** Windows, Linux 등 멀티 플랫폼 대응 폰트 및 데이터 처리 로직 보완.
- **최적화 전략:** 고성능 기기(FP32)와 초저사양 기기(INT8)를 위한 맞춤형 배포 전략 수립.

### ④ 통합 멀티태스크 진단 엔진
- **구조:** 하나의 공유 백본(Backbone)과 여러 개의 질환별 진단 헤드(Heads) 결합.
- **성과:** 뇌졸중 위험도, 치매 위험도, 보행 규칙성 점수를 **동시에 실시간 출력**하는 고효율 엔진 검증 완료.
